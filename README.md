# Mission Control Dashboard

A centralized dashboard for managing automation agents and projects.

## Status: 🚧 In Development

This dashboard is currently in early development. Core features are being built.

## What's Planned

- Agent status monitoring
- Task management system
- Project overview dashboard
- Real-time updates
- Log viewer
- Performance metrics
- Agent orchestration controls

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Supabase account (reusing `whiteboard-app` project)

### Installation

```bash
git clone https://github.com/WingsClaw/mission-control-dashboard.git
cd mission-control-dashboard
npm install
npm run dev
```

## Tech Stack

- **Frontend:** Next.js 14 + TypeScript
- **Styling:** TailwindCSS
- **Backend:** Supabase
- **Charts:** Recharts
- **Icons:** Lucide React

## Project Structure

```
mission-control-dashboard/
├── app/           # Next.js app directory
├── components/    # React components
├── lib/          # Utilities and configuration
└── public/        # Static assets
```
